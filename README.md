# walle-web

Interface web Flask pour piloter WALL-E depuis un navigateur sur le réseau local.

## Installation

```bash
pip install git+https://github.com/Buy-n-large/walle-web.git
```

## Lancement

```bash
walle-web
# Ouvrir http://wall-e.local:5000
```

## Fonctionnalités

- Joystick pour les chenilles
- Contrôle manuel de chaque servo
- Animations préprogrammées
- Lecture des sons
